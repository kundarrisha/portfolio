# 🚀 Risha Kundar — Portfolio

A personal portfolio website built with React + Vite, featuring an interactive cosmic alien scene, animated sections, and a clean developer aesthetic.

🔗 **Live site:** https://portfolio-one-delta-82.vercel.app

## ✨ Features

- **Cosmic Hero Section** — Alien in UFO with orbiting tech planets (Machine Learning, Computer Vision, Python, React, Full-Stack) and twinkling starfield
- **Particle Background** — WebGL-powered floating particles across the entire page
- **Click Sparks** — Gold spark effect on every click anywhere on the page
- **BlurText Animation** — Letter-by-letter name reveal on load
- **Smooth Scroll Navigation** — Fixed navbar with anchor scroll to all sections
- **Projects Section** — Cards with hover tilt and live/GitHub links
- **Skills Section** — Animated skill tags with hover effects
- **About Section** — Personal info with scroll-reveal animations
- **Contact Section** — Links to email, LinkedIn, and GitHub

## 🛠️ Tech Stack

- **Framework:** React 19 + Vite
- **Animations:** Framer Motion
- **3D/WebGL:** OGL, Three.js, @react-three/fiber
- **Styling:** Inline styles + CSS
- **Deployment:** Vercel

## 🚀 Running Locally

```bash
git clone https://github.com/kundarrisha/portfolio.git
cd portfolio
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.
